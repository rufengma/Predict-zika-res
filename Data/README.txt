####################################################################
Readme file describing contents of the 'Data' folder
####################################################################

> CSV files
	> flavi_mos1234: contains flavivirus vector status of mosquitos. Status can vary from 1 to 4 based on confidence of determining method (see 'Matrix Legend' tab of ../Mos-flav data.xlsx')
	> flavi_prim01: binary status of primates being flavivirus reservoirs <original version, don't use>
	> flavi_prim01new: same as flavi_prim01, but most updated. Use this
	> FlaviData: covariate data for all flaviviruses
	> mosquito: covariate data for all mosquito species
	> prim: covariate data for all primates
	> primates flavi: prim with flavLabel = flaviVirus reservoir status and ZLabel = Zoonosis reservoir status added

> XLSX files
	> Flavivirus traits + key: Flavivirus traits data- not used in modelling right now.
	> Mos-flav data: Auxiliary information on mosquitos- used in mosquito model: see supplementary doc for results.
